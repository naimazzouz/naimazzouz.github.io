Cours – Fonction affine

1. Définition

La fonction affine est définie par \( f(x) = ax + b \)
• \(a\) = coefficient directeur (pente de la droite)
• \(b\) = ordonnée à l'origine (valeur en x=0)

2. Représentation graphique

La courbe est une droite d'équation \(y = ax + b\).

  • Si \(a > 0\) : droite montante
  • Si \(a < 0\) : droite descendante
  • Si \(a = 0\) : droite horizontale (fonction constante)

3. Tracé d'une droite

Il suffit de deux points :
1. Calculer un point : prendre x=0 → y=b
2. Calculer un autre point : prendre x=1 → y=a+b

4. Intersection avec l'axe des x

Résoudre \(ax+b=0\) → \(x = -b/a\) (si \(a \ne 0\))

5. Pente (taux de variation)

\[ a = \frac{y_2 - y_1}{x_2 - x_1} \]Exemple : Si A(1;5) et B(3;11) : \(a=\frac{11-5}{3-1}=3\)
