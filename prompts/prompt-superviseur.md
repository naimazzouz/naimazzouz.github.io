# Prompt superviseur — Générer du contenu

Avant de générer une page, dans cet ordre :

1. **Programme officiel** → lire le fichier `.md` dans `/pdf/` pour le niveau concerné — ne pas introduire de hors-programme
2. **Prompt de la page** → consulter le prompt correspondant dans `/prompts/` (`prompt-cours.md`, `prompt-exercices.md`, `prompt-ds.md`, etc.)
3. **Règles visuelles** → `prompts/regles-communes.md` (données uniquement, tableaux, animations, SVG)
4. **Contextes professionnels** → prompt filière si applicable (`prompt-filiere-2mama.md`, `prompt-filiere-ticcer.md`, etc.) — jamais les sigles ICCER, ERA, MAMA dans le contenu
5. **Structure technique** → `CLAUDE.md` (thèmes couleur, chemins, classes CSS)
