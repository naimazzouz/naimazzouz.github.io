Cours – Notion de fonction

1. Définition

Une fonction est un processus qui à chaque valeur de \(x\) (variable) associe une seule valeur \(f(x)\) (image).

Notation : \(f : x \mapsto f(x)\)
Exemple : \(f(x) = 2x + 3\)   →   \(f(4) = 2\times4+3 = 11\)

2. Ensemble de définition

C'est l'ensemble des valeurs de \(x\) pour lesquelles \(f(x)\) est définie.
Attention : Pas de division par 0. Pas de racine carrée d'un négatif.

3. Tableau de valeurs

x-2-1012
f(x)=2x+3-11357

4. Courbe représentative

La courbe représentative (ou graphe) est l'ensemble des points \((x ; f(x))\) dans un repère.

5. Lecture graphique

Image : à partir de x, lire f(x) sur l'axe des ordonnées.
Antécédent : à partir de y, trouver le(s) x sur l'axe des abscisses.
