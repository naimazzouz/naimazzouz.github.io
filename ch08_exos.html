Exercice 3 Sens de variation et comparaison ★★ Intermédiaire

Sans calculer, utiliser les variations des fonctions pour répondre aux questions.

1. Comparer (f(7)) et (f(-8)) pour (f(x)=x^2). Puis vérifier par le calcul.
2. Comparer (g(3)) et (g(6)) pour (g(x)=dfrac{1}{x}). Que peut-on dire sur les variations ?
3. Sachant que (h) est croissante, ranger dans l'ordre croissant : (h(5)), (h(0{,}5)), (h(12)), (h(1)).
4. Vrai ou Faux ? « Pour tout réel (x), on a (x^2 geq 0). » Justifier.

1. (f(-8) = 64) et (f(7) = 49), donc (f(7) < f(-8)). Explication : (-8) est plus loin de (0) que (7) ((|-8|=8>7=|7|)), et (f) est croissante sur ([0;+infty[) en valeur absolue.

2. (g(3) = frac{1}{3} approx 0{,}33) et (g(6) = frac{1}{6} approx 0{,}17). Donc (g(6) < g(3)). Comme (3 < 6) et (g(6) < g(3)), (g) est bien décroissante sur (]0;+infty[).

3. (h) est croissante donc l'ordre des images suit l'ordre des (x) : (0{,}5 < 1 < 5 < 12), donc (h(0{,}5) < h(1) < h(5) < h(12)).

4. Vrai. Le carré d'un réel est toujours positif ou nul : (x^2 = (x)(x) geq 0). L'égalité a lieu uniquement pour (x=0).

Exercice 4 Résoudre des équations ★★ Intermédiaire

Résoudre les équations suivantes dans (mathbb{R}).

1. (x^2 = 16)
2. (x^2 = 7) (donner les solutions exactes puis approchées)
3. (dfrac{1}{x} = 4)
4. (sqrt{x} = 3)
5. (sqrt{x} = -2) (attention !)

1. (x^2=16 Rightarrow x=4) ou (x=-4). Solutions : ({-4,;,4}).

2. (x^2=7 Rightarrow x=sqrt{7}) ou (x=-sqrt{7}). Approché : (x approx 2{,}65) ou (x approx -2{,}65).

3. (dfrac{1}{x}=4 Rightarrow 1=4x Rightarrow x=dfrac{1}{4}=0{,}25). Solution : (left{dfrac{1}{4} ight}).

4. (sqrt{x}=3 Rightarrow x=3^2=9). Solution : ({9}).

5. (sqrt{x}=-2) : pas de solution. En effet, (sqrt{x} geq 0) toujours, donc (sqrt{x}) ne peut pas valoir (-2). (mathcal{S} = emptyset).

Exercice 5 Domaine de définition ★★ Intermédiaire

Pour chaque expression, préciser les valeurs de (x) pour lesquelles elle est définie.

1. (sqrt{x-3})
2. (dfrac{1}{x+2})
3. (sqrt{2x+6})
4. (dfrac{1}{x^2-9})

1. On doit avoir (x-3 geq 0), soit (x geq 3). Domaine : ([3,;,+infty[).

2. On doit avoir (x+2 eq 0), soit (x eq -2). Domaine : (mathbb{R}setminus{-2}).

3. On doit avoir (2x+6 geq 0), soit (x geq -3). Domaine : ([-3,;,+infty[).

4. On doit avoir (x^2-9 eq 0), soit (x eq 3) et (x eq -3). Domaine : (mathbb{R}setminus{-3,;,3}).

Exercice 6 Problème – Aire d'un carré ★★★ Avancé

Un carré a un côté de longueur (c) cm. Son aire est donnée par (A(c) = c^2).

1. Calculer l'aire d'un carré de côté 7 cm.
2. Quel est le côté d'un carré d'aire 144 cm² ?
3. On dispose de 50 cm de fil pour former le périmètre d'un carré. Quelle est l'aire de ce carré ?
4. L'aire d'un carré est multipliée par 9. Par quel facteur le côté est-il multiplié ?

1. (A(7) = 7^2 = 49 ext{ cm}^2).

2. (c^2 = 144 Rightarrow c = sqrt{144} = 12) cm (on prend la racine positive car (c > 0)).

3. Périmètre (= 4c = 50), donc (c = 12{,}5) cm. Aire (= (12{,}5)^2 = 156{,}25 ext{ cm}^2).

4. Si l'aire est multipliée par 9 : nouvelle aire (= 9c^2 = (3c)^2). Le côté est donc multiplié par (3 = sqrt{9}).
