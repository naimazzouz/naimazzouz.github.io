Fiche résumé — Lumière, couleurs et photodétecteurs

Chapitre 14 | Seconde Bac Pro | Physique-Chimie

Nature de la lumière

  • Onde électromagnétique
  • Spectre visible : 380 nm (violet) à 780 nm (rouge)
  • 1 nm = \(10^{-9}\) m
\( c = \lambda \times f \)

\(c = 3 \times 10^8\) m/s, \(\lambda\) en m, \(f\) en Hz

Spectre visible

  • Violet : 380-450 nm
  • Bleu : 450-495 nm
  • Vert : 495-570 nm
  • Jaune : 570-590 nm
  • Orange : 590-620 nm
  • Rouge : 620-780 nm

Lumière blanche = superposition de toutes les couleurs.

Synthèse additive (RGB)

  • Primaires : Rouge + Vert + Bleu
  • R + G = Jaune
  • R + B = Magenta
  • G + B = Cyan
  • R + G + B = Blanc

Utilisée pour les écrans et éclairages LED RGB.

Synthèse soustractive (CMJ)

  • Primaires : Cyan + Magenta + Jaune
  • Chaque filtre absorbe une composante
  • C + M + J = Noir (en théorie)

Utilisée pour l'impression et les pigments (peinture, lasure bois).

Photodétecteurs

  • LDR : résistance diminue avec la lumière (éclairage automatique)
  • Photodiode : rapide, capteurs de présence sur machines
  • Cellule photovoltaïque : convertit lumière en électricité
  • Capteur CCD/CMOS : imagerie (caméra de contrôle qualité)

Éclairement et loi en 1/d²

\( E = \dfrac{I}{d^2} \)
  • \(E\) : éclairement en lux (lx)
  • \(I\) : intensité lumineuse en candelas (cd)
  • \(d\) : distance en m
  • Atelier : 500 lx recommandés (norme)
Piège 1 : Confondre synthèse additive et soustractive. Additive (lumières) : R+G+B = blanc. Soustractive (pigments) : C+M+J = noir.
Piège 2 : Oublier de convertir les nm en m pour utiliser \(c = \lambda \times f\). Exemple : 550 nm = \(550 \times 10^{-9}\) m.
Piège 3 : Penser qu'un objet rouge « produit » de la lumière rouge. En réalité, il absorbe toutes les couleurs sauf le rouge qu'il diffuse.
Astuce : Pour la finition du bois, choisir une source avec un IRC (indice de rendu des couleurs) supérieur ou égal à 90 pour percevoir les teintes fidèlement.
Astuce : La loi en \(1/d^2\) signifie que si on double la distance, l'éclairement est divisé par 4. Rapprocher la source améliore beaucoup l'éclairage.

Résumé express — Méthode type

  1. Identifier le type de synthèse : additive (lumières, écrans) ou soustractive (pigments, impression).
  2. Pour calculer \(f\) ou \(\lambda\) : utiliser \(c = \lambda \times f\) avec \(\lambda\) en mètres.
  3. Pour l'éclairement : appliquer \(E = I/d^2\), vérifier que la distance est en mètres.
  4. Pour choisir un photodétecteur : LDR pour l'éclairage automatique, photodiode pour la détection rapide.
  5. Vérifier les unités : \(\lambda\) en m (ou nm pour le spectre), \(f\) en Hz, \(E\) en lux.
