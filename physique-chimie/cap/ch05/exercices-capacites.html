Capacité 1 — Déterminer la période et la fréquence d'un son pur

Exercice 1.1 — Lecture d'oscillogramme

Pour chacun des oscillogrammes décrits ci-dessous, calculez la période puis la fréquence :

SignalBase de tempsPériode (div)T (ms)f (Hz)
Diapason0,5 ms/div4,5
Alarme atelier0,2 ms/div2,0
Bruit toupie0,1 ms/div1,5
Correction : Diapason : T = 4,5 × 0,5 = 2,25 ms → f = 1/0,00225 ≈ 444 Hz (proche du La 440 Hz) / Alarme : T = 2 × 0,2 = 0,4 ms → f = 2500 Hz / Toupie : T = 1,5 × 0,1 = 0,15 ms → f ≈ 6667 Hz.

Exercice 1.2 — Classement grave/aigu

Classez ces sons du plus grave au plus aigu et identifiez ceux qui sont hors du domaine audible :

SourceFréquence (Hz)Rang (grave → aigu)Domaine
Vibration d'une poutre8
Voix masculine basse100
Sifflement de scie3000
Contrôle ultrasonique40 000
Alarme haute fréquence8000
Correction : 8 Hz → infrasون (non audible) / 100 Hz → grave (audible) / 3000 Hz → aigu (audible) / 8000 Hz → très aigu (audible) / 40 000 Hz → ultrason (non audible). Classement audibles : 100 Hz < 3000 Hz < 8000 Hz.

Capacité 3 — Comparer les atténuations phoniques

Exercice 3.1 — Choix du matériau isolant

On cherche à réduire le bruit d'une scie circulaire (102 dB) en dessous de 75 dB dans la salle adjacente. Quatre matériaux sont disponibles :

MatériauAtténuationNiveau résiduelSuffisant ?
Mousse fine (2 cm)10 dB
Laine de verre (8 cm)22 dB
Laine de roche (12 cm)30 dB
Double paroi (béton + air + béton)38 dB
Correction : Mousse : 92 dB, non / Laine de verre : 80 dB, non / Laine de roche : 72 dB, oui / Double paroi : 64 dB, oui. Il faut au moins 27 dB d'atténuation.
