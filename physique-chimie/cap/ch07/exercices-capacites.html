Capacité 1 — Vérifier la loi de réflexion

Exercice 1.1 — Applications de la loi de réflexion

  1. Un rayon lumineux arrive sur une glace (miroir plan) avec un angle d'incidence de 25°. Calculez l'angle de réflexion.
  2. Si l'angle d'incidence est nul (rayon perpendiculaire au miroir), où repart le rayon réfléchi ?
  3. Un artisan utilise un miroir incliné à 45° pour renvoyer un faisceau laser horizontal vers le plafond. Quel est l'angle d'incidence sur le miroir ? Quel est l'angle entre le rayon incident et le rayon réfléchi ?
  4. Sur un chantier, un miroir plan est utilisé pour voir autour d'un coin de mur (angle de 90°). Quel angle d'inclinaison le miroir doit-il avoir pour renvoyer le faisceau à 90° ?
Correction :
  1. r = 25°.
  2. Le rayon repart exactement dans la direction inverse (r = 0°).
  3. Le faisceau horizontal arrive sur le miroir à 45° (angle avec la normale verticale). r = 45°. Angle incident-réfléchi = 45° + 45° = 90°.
  4. Pour une déviation de 90°, il faut un angle d'incidence de 45° → miroir incliné à 45°.

Capacité 2 — Réfraction et indice

Exercice 2.1 — Changement de milieu

Données : n(air) = 1,00 / n(eau) = 1,33 / n(verre) = 1,50 / n(plexiglas) = 1,49 / n(diamant) = 2,42.

  1. Un rayon passe de l'air dans l'eau avec i = 40°. Se rapproche-t-il ou s'éloigne-t-il de la normale ?
  2. Un rayon passe du verre dans l'air avec i = 30°. Que se passe-t-il par rapport à la normale ?
  3. Parmi les milieux donnés, lequel dévie le plus la lumière ? Justifiez.
  4. Un signalétiste coupe du plexiglas. Le rayon laser passe de l'air dans le plexiglas. Décrivez la déviation du rayon.
Correction :
  1. Air → eau (n plus élevé) : se rapproche de la normale (angle de réfraction < 40°).
  2. Verre → air (n plus faible) : s'éloigne de la normale (angle de réfraction > 30°).
  3. Le diamant (n = 2,42) : indice le plus élevé → déviation la plus importante.
  4. Air → plexiglas (n = 1,49 > 1,00) : le rayon se rapproche de la normale à l'entrée dans le plexiglas.

Capacité 3 — Décomposer la lumière blanche, spectre visible, IR et UV

Exercice 3.1 — Spectre et rayonnements

  1. Classez ces rayonnements par longueur d'onde croissante : infrarouges, rouge, ultraviolets, violet, vert.
  2. Un lampe UV industrielle émet à 254 nm. Est-ce dans le visible ? Quel est le danger ?
  3. Un panneau solaire infrarouge émet à 900 nm. Est-ce visible ? Quel effet thermique cela peut-il avoir sur la peau ?
  4. Pour voir les vraies couleurs d'un vernis bois, doit-on travailler sous lampe à incandescence (blanc chaud, 2700 K) ou sous éclairage LED blanc (5000 K) ? Justifiez.
Correction :
  1. UV (< 380 nm) < Violet (~400 nm) < Vert (~530 nm) < Rouge (~680 nm) < IR (> 780 nm).
  2. 254 nm : hors du visible (UV), non perceptible. Danger : brûlures cutanées graves, lésions oculaires (conjonctivite, cataracte).
  3. 900 nm : IR, non visible. Effet thermique : chaleur ressentie sur la peau, risque de brûlure par IR intenses.
  4. LED 5000 K (proche lumière du jour) : spectre complet → couleurs fidèles. Incandescence 2700 K : trop jaune, fausse la perception des teintes bleues et vertes.

  1. Un écran de signalétique LED affiche du bleu pur. Quelles LED sont allumées ?
  2. En synthèse soustractive (peintures), R + B + J = noir. Pourquoi le résultat est-il différent de la synthèse additive ?
Correction :
  1. Rouge / Jaune / Cyan / Magenta / Blanc / Noir.
  2. Seule la LED bleue est allumée (R = 0, V = 0, B = 100%).
  3. La synthèse soustractive (pigments) absorbe des longueurs d'onde. En mélangeant les pigments, chaque couleur absorbe une partie du spectre : au total, tout est absorbé → noir. La synthèse additive ajoute des lumières : l'addition de toutes les couleurs donne du blanc.
