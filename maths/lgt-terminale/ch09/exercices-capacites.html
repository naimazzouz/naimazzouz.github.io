C1 — Utiliser les propriétés du logarithme pour transformer une écriture

Exercice 1

Simplifier les expressions suivantes :

  1. \(\ln(e^4) - 2\ln(e)\)
  2. \(\ln 8 + \ln 5 - \ln 10\)
  3. \(\ln\sqrt{e^6}\)
  4. \(3\ln 2 - \ln 4 + \ln 6\)
  1. \(4 - 2 = 2\).
  2. \(\ln\frac{8\times 5}{10} = \ln 4 = 2\ln 2\).
  3. \(\ln(e^3) = 3\).
  4. \(\ln 8 - \ln 4 + \ln 6 = \ln\frac{8\times 6}{4} = \ln 12\).
Exercice 2

Exprimer en fonction de \(\ln 2\) et \(\ln 3\) :

  1. \(\ln 6\)
  2. \(\ln 72\)
  3. \(\ln\frac{9}{16}\)
  4. \(\ln 0{,}75\)
  1. \(\ln 6 = \ln(2\times 3) = \ln 2+\ln 3\).
  2. \(\ln 72 = \ln(8\times 9) = 3\ln 2+2\ln 3\).
  3. \(\ln\frac{9}{16} = 2\ln 3-4\ln 2\).
  4. \(\ln 0{,}75 = \ln\frac{3}{4} = \ln 3-2\ln 2\).
Exercice 3

Dériver les fonctions suivantes :

  1. \(f(x) = \ln(x^2+4x)\)
  2. \(g(x) = x^2\ln x\)
  3. \(h(x) = \frac{\ln x}{x^2}\)
  4. \(k(x) = \ln(\cos x)\) sur \(]-\frac{\pi}{2};\frac{\pi}{2}[\)
  1. \(f'(x) = \frac{2x+4}{x^2+4x} = \frac{2(x+2)}{x(x+4)}\).
  2. \(g'(x) = 2x\ln x+x^2\cdot\frac{1}{x} = 2x\ln x+x = x(2\ln x+1)\).
  3. \(h'(x) = \frac{\frac{1}{x}\cdot x^2-\ln x\cdot 2x}{x^4} = \frac{x-2x\ln x}{x^4} = \frac{1-2\ln x}{x^3}\).
  4. \(k'(x) = \frac{-\sin x}{\cos x} = -\tan x\).
Exercice 4

Déterminer les limites suivantes :

  1. \(\displaystyle\lim_{x \to +\infty} \frac{(\ln x)^2}{x}\)
  2. \(\displaystyle\lim_{x \to 0^+} \sqrt{x}\ln x\)
  3. \(\displaystyle\lim_{x \to +\infty} \left(2x-3\ln x\right)\)
  4. \(\displaystyle\lim_{x \to 1} \frac{\ln x}{x-1}\)
  1. On pose \(X=\ln x\) : \(\frac{X^2}{e^X}\to 0\). Limite = 0.
  2. On pose \(x=e^{-t}\), \(t\to+\infty\) : \(e^{-t/2}\times(-t)=-te^{-t/2}\to 0\). Limite = 0.
  3. \(2x-3\ln x = x\left(2-\frac{3\ln x}{x}\right)\to x\times 2\to+\infty\).
  4. C'est \((\ln)'(1)=\frac{1}{1}=1\). Limite = 1.
Exercice 5

Simplifier les expressions suivantes :

  1. \(\displaystyle\ln\!\left(\frac{e^2\sqrt{e}}{e^{-3}}\right)\)
  2. \(\displaystyle\ln\!\left(\frac{27}{8}\right)\) en fonction de \(\ln 2\) et \(\ln 3\)
  1. \(\ln\!\left(\frac{e^2\sqrt{e}}{e^{-3}}\right) = \ln(e^2)+\ln(e^{1/2})-\ln(e^{-3}) = 2+\frac{1}{2}-(-3) = \frac{11}{2}\).
  2. \(\ln\!\left(\frac{27}{8}\right) = \ln 27 - \ln 8 = \ln(3^3)-\ln(2^3) = 3\ln 3 - 3\ln 2\).
Exercice 6

Soit \(f(x) = \ln(\sin x)\) définie sur \(]0;\pi[\).

  1. Justifier que \(f\) est bien définie sur \(]0;\pi[\).
  2. Calculer \(f'(x)\).
  3. Déterminer les variations de \(f\) sur \(]0;\pi[\).
  4. Calculer les limites de \(f\) en \(0^+\) et en \(\pi^-\).
  1. Sur \(]0;\pi[\), \(\sin x > 0\), donc \(\ln(\sin x)\) est défini. ✓
  2. Par dérivation d'une composée : \(f'(x) = \frac{\cos x}{\sin x} = \frac{1}{\tan x}\) (cotangente).
  3. \(f'(x)>0 \Leftrightarrow \cos x > 0 \Leftrightarrow x\in\left]0;\frac{\pi}{2}\right[\). \(f'(x)<0\) sur \(\left]\frac{\pi}{2};\pi\right[\).
    \(f\) est croissante sur \(\left]0;\frac{\pi}{2}\right]\) et décroissante sur \(\left[\frac{\pi}{2};\pi\right[\). Maximum en \(x=\frac{\pi}{2}\) : \(f\!\left(\frac{\pi}{2}\right)=\ln 1 = 0\).
  4. \(\lim_{0^+}\sin x = 0^+\), donc \(\lim_{0^+}f = -\infty\). De même \(\lim_{\pi^-}\sin x = 0^+\), donc \(\lim_{\pi^-}f = -\infty\).

C2 — Résoudre une équation ou une inéquation

Exercice 7

Résoudre dans \(\mathbb{R}\) :

  1. \(\ln(3x+2) = \ln(x+6)\)
  2. \(\ln(x-1) + \ln(x+1) = \ln 8\)
  3. \(2\ln x - \ln(x+6) = 0\)
  1. Conditions : \(3x+2>0\) et \(x+6>0\), soit \(x>-\frac{2}{3}\). \(3x+2=x+6\), \(2x=4\), \(x=2\). ✓.
  2. Conditions : \(x>1\). \(\ln[(x-1)(x+1)]=\ln 8\), \(x^2-1=8\), \(x^2=9\), \(x=3\) (on rejette \(-3\)).
  3. Condition : \(x>0\). \(\ln(x^2)=\ln(x+6)\), \(x^2=x+6\), \(x^2-x-6=0\), \((x-3)(x+2)=0\). \(x=3\) (\(x=-2\) rejeté).
Exercice 8

Résoudre les inéquations :

  1. \(\ln(2x+1) > 0\)
  2. \(\ln(x-3) \leqslant 1\)
  3. \(\ln x^2 > 2\ln 3\)
  1. Condition : \(x>-\frac{1}{2}\). \(\ln(2x+1)>0=\ln 1 \Leftrightarrow 2x+1>1 \Leftrightarrow x>0\). Solution : \(x\in]0;+\infty[\).
  2. Condition : \(x>3\). \(x-3\leqslant e^1=e\), \(x\leqslant e+3\). Solution : \(x\in]3;e+3]\).
  3. Condition : \(x\neq 0\). \(\ln x^2>2\ln 3=\ln 9\), \(x^2>9\), \(|x|>3\). Solution : \(x\in]-\infty;-3[\cup]3;+\infty[\).
Exercice 9

Résoudre :

  1. \(e^{3x-1} = 7\)
  2. \(e^{2x} - 5e^x + 6 = 0\)
  3. \(e^x > x + 1\) (discuter)
  1. \(3x-1=\ln 7\), \(x=\frac{1+\ln 7}{3}\).
  2. On pose \(X=e^x>0\). \(X^2-5X+6=0\), \((X-2)(X-3)=0\). \(X=2\) ou \(X=3\).
    \(e^x=2\Rightarrow x=\ln 2\). \(e^x=3\Rightarrow x=\ln 3\).
  3. On a montré (ch07) que \(e^x\geqslant 1+x\) pour tout \(x\). L'égalité n'a lieu qu'en \(x=0\). Donc \(e^x>x+1\) pour tout \(x\neq 0\). Solution : \(\mathbb{R}\setminus\{0\}\).
Exercice 10

Résoudre \(3^x = 5\).

\(\ln(3^x) = \ln 5\), \(x\ln 3 = \ln 5\), \(x = \frac{\ln 5}{\ln 3}\approx 1{,}465\).

Exercice 11

Étude complète

Soit \(f(x) = x^2 - 4\ln x\) définie sur \(]0;+\infty[\).

  1. Limites en \(0^+\) et \(+\infty\).
  2. Variations de \(f\).
  3. Montrer que l'équation \(f(x) = 0\) admet exactement deux solutions.
  1. \(\lim_{0^+}f = 0-4(-\infty)=+\infty\). \(\lim_{+\infty}f = +\infty\) (\(x^2\) domine).
  2. \(f'(x) = 2x-\frac{4}{x} = \frac{2x^2-4}{x} = \frac{2(x^2-2)}{x}\). \(f'=0\) pour \(x=\sqrt{2}\) (\(x>0\)).
    \(f'<0\) sur \(]0;\sqrt{2}[\), \(f'>0\) sur \(]\sqrt{2};+\infty[\). Min en \(x=\sqrt{2}\) : \(f(\sqrt{2}) = 2-4\times\frac{1}{2}\ln 2 = 2-2\ln 2\approx 0{,}61\).
  3. Le minimum vaut \(2-2\ln 2\approx 0{,}61>0\). Donc \(f(x)>0\) pour tout \(x>0\). L'équation \(f(x)=0\) n'admet aucune solution.
    (Correction : le minimum étant positif, \(f\) ne s'annule pas.)
Exercice 12

Problème de synthèse

Soit \(f(x) = \ln x + \frac{1}{x} - 1\) définie sur \(]0;+\infty[\).

  1. Étudier les variations de \(f\).
  2. En déduire le signe de \(f(x)\) et montrer que pour tout \(x>0\) : \(\ln x \geqslant 1-\frac{1}{x}\).
  3. En déduire que pour tout \(x>0\) : \(\frac{1}{x+1}\leqslant\ln\frac{x+1}{x}\leqslant\frac{1}{x}\).
  1. \(f'(x) = \frac{1}{x}-\frac{1}{x^2}=\frac{x-1}{x^2}\). \(f'=0\) pour \(x=1\). \(f'<0\) sur \(]0;1[\), \(f'>0\) sur \(]1;+\infty[\).
    Min en \(x=1\) : \(f(1)=0+1-1=0\).
  2. \(f(x)\geqslant f(1)=0\), donc \(\ln x+\frac{1}{x}-1\geqslant 0\), soit \(\ln x\geqslant 1-\frac{1}{x}\) pour tout \(x>0\). □
  3. On applique \(\ln x\leqslant x-1\) avec \(x\) remplacé par \(\frac{x+1}{x}=1+\frac{1}{x}\) : \(\ln\frac{x+1}{x}\leqslant\frac{1}{x}\).
    On applique \(\ln x\geqslant 1-\frac{1}{x}\) avec \(x\) remplacé par \(\frac{x+1}{x}\) : \(\ln\frac{x+1}{x}\geqslant 1-\frac{x}{x+1}=\frac{1}{x+1}\). □
Exercice 13

Résoudre l'équation \(\ln(x^2-3x+2) = 0\).

Domaine : il faut \(x^2-3x+2>0\). On factorise : \(x^2-3x+2=(x-1)(x-2)\). Ce produit est strictement positif pour \(x\in]-\infty;1[\cup]2;+\infty[\).

Équation : \(\ln(x^2-3x+2)=0 \Leftrightarrow x^2-3x+2=1 \Leftrightarrow x^2-3x+1=0\).

\(\Delta=9-4=5\). \(x=\frac{3\pm\sqrt{5}}{2}\).

\(x_1=\frac{3-\sqrt{5}}{2}\approx 0{,}38\). Comme \(x_1<1\), \(x_1\) appartient au domaine. ✓

\(x_2=\frac{3+\sqrt{5}}{2}\approx 2{,}62\). Comme \(x_2>2\), \(x_2\) appartient au domaine. ✓

Solutions : \(S=\left\{\frac{3-\sqrt{5}}{2};\ \frac{3+\sqrt{5}}{2}\right\}\).

Exercice 14

Résoudre l'inéquation \(2\ln(x) + \ln(x-1) \leqslant \ln(4x)\).

Domaine : il faut \(x>0\) et \(x-1>0\) et \(4x>0\), soit \(x>1\).

On transforme le membre de gauche : \(2\ln x+\ln(x-1)=\ln(x^2)+\ln(x-1)=\ln[x^2(x-1)]\).

L'inéquation devient : \(\ln[x^2(x-1)]\leqslant\ln(4x)\).

Comme \(\ln\) est croissante : \(x^2(x-1)\leqslant 4x\). Comme \(x>1>0\), on divise par \(x\) :

\(x(x-1)\leqslant 4\), soit \(x^2-x-4\leqslant 0\).

\(\Delta=1+16=17\). Racines : \(x=\frac{1\pm\sqrt{17}}{2}\). On a \(\frac{1-\sqrt{17}}{2}\approx -1{,}56\) et \(\frac{1+\sqrt{17}}{2}\approx 2{,}56\).

\(x^2-x-4\leqslant 0\) pour \(x\in\left[\frac{1-\sqrt{17}}{2};\ \frac{1+\sqrt{17}}{2}\right]\).

En intersectant avec le domaine \(x>1\) : \(S=\left]1;\ \frac{1+\sqrt{17}}{2}\right]\).

Exercice 15

Étude complète

Soit \(f(x) = (\ln x)^2 - 2\ln x\) définie sur \(]0;+\infty[\).

  1. Calculer les limites de \(f\) en \(0^+\) et en \(+\infty\).
  2. Calculer \(f'(x)\) et étudier les variations de \(f\).
  3. Déterminer le minimum de \(f\).
  4. Résoudre \(f(x)=0\) et en déduire le signe de \(f\).
  1. En \(0^+\) : \(\ln x\to -\infty\), donc \((\ln x)^2\to +\infty\) et \(-2\ln x\to +\infty\). \(\lim_{0^+}f=+\infty\).
    En \(+\infty\) : \(\ln x\to +\infty\), donc \((\ln x)^2\to +\infty\) et domine \(-2\ln x\). \(\lim_{+\infty}f=+\infty\).
  2. On pose \(u=\ln x\), alors \(f(x)=u^2-2u\) avec \(u'=\frac{1}{x}\).
    \(f'(x)=(2\ln x - 2)\times\frac{1}{x}=\frac{2(\ln x-1)}{x}\).
    \(f'(x)=0 \Leftrightarrow \ln x=1 \Leftrightarrow x=e\). Pour \(x>0\) : \(f'<0\) sur \(]0;e[\), \(f'>0\) sur \(]e;+\infty[\).
  3. Minimum en \(x=e\) : \(f(e)=(\ln e)^2-2\ln e = 1-2=-1\).
  4. \(f(x)=0 \Leftrightarrow (\ln x)^2-2\ln x=0 \Leftrightarrow \ln x(\ln x-2)=0\).
    \(\ln x=0 \Rightarrow x=1\). \(\ln x=2 \Rightarrow x=e^2\).
    Signe : \(f(x)\geqslant 0\) sur \(]0;1]\cup[e^2;+\infty[\) et \(f(x)\leqslant 0\) sur \([1;e^2]\).
