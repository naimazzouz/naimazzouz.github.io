Exercice 1 – Contrôle qualité en menuiserie 8 points

Un atelier de menuiserie fabrique des panneaux en bois. On contrôle 200 panneaux selon deux critères : dimensions conformes (D) et finition conforme (F). On obtient le tableau suivant :

F (finition conforme)\(\overline{F}\) (finition non conforme)Total
D (dimensions conformes)15018168
\(\overline{D}\) (dimensions non conformes)122032
Total16238200

On prélève un panneau au hasard parmi les 200.

1. APP Quelle est la probabilité que le panneau ait des dimensions conformes ? (1 pt)

2. REA Calculer la probabilité que le panneau soit conforme sur les deux critères (D et F). (1 pt)

3. REA Calculer la probabilité que le panneau ne soit conforme sur aucun des deux critères. (1 pt)

4. ANA Sachant que le panneau a des dimensions conformes, quelle est la probabilité que sa finition soit non conforme ? (2 pts)

5. COM Compléter l'arbre de probabilités ci-dessous avec les probabilités conditionnelles. (3 pts)

                        ┌── F : P(F|D) = ___
             ┌── D : ___┤
             │          └── F̄ : P(F̄|D) = ___
  Panneau ───┤
             │          ┌── F : P(F|D̄) = ___
             └── D̄ : ___┤
                        └── F̄ : P(F̄|D̄) = ___

1. \(P(D) = \dfrac{168}{200} = 0{,}84\)

2. \(P(D \cap F) = \dfrac{150}{200} = 0{,}75\)

3. \(P(\overline{D} \cap \overline{F}) = \dfrac{20}{200} = 0{,}10\)

4. \(P_D(\overline{F}) = \dfrac{18}{168} = \dfrac{3}{28} \approx 0{,}107\). Sachant que le panneau a des dimensions conformes, il y a environ 10,7 % de chances que sa finition soit non conforme.

5. Arbre complété :

  • \(P(D) = 0{,}84\) ; \(P(\overline{D}) = 0{,}16\)
  • \(P_D(F) = \dfrac{150}{168} \approx 0{,}893\) ; \(P_D(\overline{F}) = \dfrac{18}{168} \approx 0{,}107\)
  • \(P_{\overline{D}}(F) = \dfrac{12}{32} = 0{,}375\) ; \(P_{\overline{D}}(\overline{F}) = \dfrac{20}{32} = 0{,}625\)
Exercice 2 – Espérance et prise de décision 12 points

Une entreprise de chauffage installe des pompes à chaleur. Après installation, chaque pompe peut présenter :

  • un défaut mineur (réglage) avec une probabilité de \(0{,}15\) ;
  • un défaut majeur (pièce défectueuse) avec une probabilité de \(0{,}05\) ;
  • aucun défaut sinon.

1. APP Quelle est la probabilité qu'une pompe à chaleur n'ait aucun défaut ? (1 pt)

2. COM Compléter l'arbre de probabilités ci-dessous. (2 pts)

             ┌── Défaut mineur : ___
  Pompe ─────┼── Défaut majeur : ___
             └── Aucun défaut : ___

Le coût de réparation est de :

  • 80 € pour un défaut mineur ;
  • 450 € pour un défaut majeur ;
  • 0 € si aucun défaut.

On note \(X\) la variable aléatoire représentant le coût de réparation d'une pompe.

3. APP Quelles sont les valeurs possibles de \(X\) ? (1 pt)

4. REA Compléter le tableau de la loi de probabilité de \(X\) : (2 pts)

\(x_i\)080450
\(P(X = x_i)\)

5. REA Calculer l'espérance mathématique \(E(X)\). (2 pts)

6. COM Interpréter le résultat dans le contexte de l'entreprise. (1 pt)

7. ANA L'entreprise envisage un contrôle systématique coûtant 40 € par pompe, qui élimine tous les défauts mineurs et réduit les défauts majeurs à une probabilité de \(0{,}02\). Le contrôle est-il rentable ? Justifier par le calcul. (3 pts)

1. \(P(\text{aucun défaut}) = 1 - 0{,}15 - 0{,}05 = 0{,}80\)

2. Arbre : Défaut mineur \(0{,}15\) ; Défaut majeur \(0{,}05\) ; Aucun défaut \(0{,}80\).

3. Les valeurs possibles de \(X\) sont : \(0\), \(80\) et \(450\).

4.

\(x_i\)080450
\(P(X = x_i)\)\(0{,}80\)\(0{,}15\)\(0{,}05\)

5. \(E(X) = 0 \times 0{,}80 + 80 \times 0{,}15 + 450 \times 0{,}05 = 0 + 12 + 22{,}50 = 34{,}50\) €

6. En moyenne, le coût de réparation par pompe installée est de 34,50 €. L'entreprise doit prévoir ce budget moyen dans ses charges.

7. Avec le contrôle, la nouvelle espérance est : \(E(X') = 40 + 0 \times 0{,}98 + 450 \times 0{,}02 = 40 + 9 = 49\) €. Sans contrôle : \(E(X) = 34{,}50\) €. Le contrôle coûte en moyenne \(49 - 34{,}50 = 14{,}50\) € de plus par pompe. Le contrôle n'est pas rentable dans ces conditions.
