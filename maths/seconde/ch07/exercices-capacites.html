C1 — Reconnaître une fonction et utiliser le vocabulaire

Rappel de cours

Une fonction associe à chaque valeur d'entrée une seule valeur de sortie. On note \(f(x)\) l'image de \(x\) par \(f\). L'ensemble des valeurs d'entrée autorisées est l'ensemble de définition.

Exercice 1

Pour chaque situation, dire si elle peut être modélisée par une fonction. Justifier.

  1. À chaque longueur de planche (en m) correspond un prix (en €).
  2. À chaque prénom correspond un âge.
  3. À chaque valeur de \(x\) correspond \(f(x) = 3x + 2\).
  4. À chaque nombre correspond deux de ses doubles.
  1. Oui — à chaque longueur correspond un prix unique. C'est une fonction.
  2. Non (en général) — plusieurs personnes peuvent avoir le même prénom avec des âges différents. Ce n'est pas une fonction (sauf si les prénoms sont tous distincts).
  3. Oui — la formule \(f(x) = 3x + 2\) associe à chaque réel \(x\) une valeur unique.
  4. Non — « deux de ses doubles » n'est pas une valeur unique. Une fonction associe à chaque entrée une seule sortie.

Exercice 2

Soit \(f(x) = 4x - 1\). Répondre aux questions suivantes en utilisant le vocabulaire correct :

  1. Quel est le nombre associé à \(x = 3\) ? Comment s'appelle-t-il ?
  2. Pour quel \(x\) obtient-on \(f(x) = 11\) ? Comment s'appelle ce \(x\) ?
  3. Écrire la notation fonctionnelle correcte pour "l'image de 5".
  1. \(f(3) = 4 \times 3 - 1 = 11\). Ce nombre s'appelle l'image de 3 par \(f\).
  2. \(4x - 1 = 11 \Rightarrow 4x = 12 \Rightarrow x = 3\). Ce \(x\) s'appelle un antécédent de 11 par \(f\).
  3. On écrit \(f(5)\), qui se lit "f de 5" ou "l'image de 5 par f".

Exercice 3

Un menuisier facture ses interventions selon la formule \(C(h) = 40h + 50\), où \(h\) est la durée en heures.

  1. Quelle est la variable d'entrée ? La variable de sortie ?
  2. Quel est le coût pour une intervention de 2 h ? Quel est le nom de ce résultat ?
  3. Pour quel nombre d'heures le coût est-il de 170 € ? Comment appelle-t-on ce nombre d'heures ?
  1. Variable d'entrée : \(h\) (durée en heures). Variable de sortie : \(C(h)\) (coût en euros).
  2. \(C(2) = 40 \times 2 + 50 = 130\) €. Ce résultat est l'image de 2 par \(C\).
  3. \(40h + 50 = 170 \Rightarrow 40h = 120 \Rightarrow h = 3\) h. Ce nombre est un antécédent de 170 par \(C\).

C2 — Calculer l'image d'un nombre par une fonction

Rappel de cours

Pour calculer l'image de \(a\) par \(f\) : remplacer \(x\) par \(a\) dans la formule et effectuer le calcul. On note le résultat \(f(a)\).

Exemple : si \(f(x) = 3x + 1\), alors \(f(2) = 3 \times 2 + 1 = 7\).

Exercice 4

Calculer les images suivantes pour \(f(x) = 2x + 3\) :

  1. \(f(0)\)
  2. \(f(4)\)
  3. \(f(-2)\)
  4. \(f(0{,}5)\)
  1. \(f(0) = 2 \times 0 + 3 = 3\)
  2. \(f(4) = 2 \times 4 + 3 = 11\)
  3. \(f(-2) = 2 \times (-2) + 3 = -4 + 3 = -1\)
  4. \(f(0{,}5) = 2 \times 0{,}5 + 3 = 1 + 3 = 4\)

Exercice 5

Soit \(g(x) = x^2 - 1\). Calculer :

  1. \(g(3)\)
  2. \(g(-3)\)
  3. \(g(0)\)
  4. Que remarque-t-on entre \(g(3)\) et \(g(-3)\) ?
  1. \(g(3) = 3^2 - 1 = 9 - 1 = 8\)
  2. \(g(-3) = (-3)^2 - 1 = 9 - 1 = 8\)
  3. \(g(0) = 0^2 - 1 = -1\)
  4. \(g(3) = g(-3) = 8\) : 3 et −3 ont la même image. Cela s'explique car \(x^2\) donne le même résultat pour \(x\) et \(-x\).

Exercice 6

Un artisan fabrique des cadres carrés de côté \(c\) cm. L'aire du cadre (surface totale) est donnée par \(A(c) = c^2\). Calculer l'aire pour des cadres de côté 15 cm, 20 cm et 30 cm.

\(A(15) = 15^2 = 225 \text{ cm}^2\)
\(A(20) = 20^2 = 400 \text{ cm}^2\)
\(A(30) = 30^2 = 900 \text{ cm}^2\)
Remarque : quand le côté double (15 → 30), l'aire est multipliée par 4.

C3 — Déterminer un antécédent d'un nombre par une fonction

Rappel de cours

Chercher un antécédent de \(b\) par \(f\) revient à résoudre l'équation \(f(x) = b\).

Une image est unique ; un antécédent peut ne pas exister, ou en avoir plusieurs.

Exercice 7

Soit \(f(x) = 3x - 6\). Trouver les antécédents des nombres suivants :

  1. Antécédent de 0
  2. Antécédent de 9
  3. Antécédent de −3
On résout \(f(x) = \ldots\) dans chaque cas.
  1. \(3x - 6 = 0 \Rightarrow 3x = 6 \Rightarrow x = 2\). L'antécédent de 0 est 2.
  2. \(3x - 6 = 9 \Rightarrow 3x = 15 \Rightarrow x = 5\). L'antécédent de 9 est 5.
  3. \(3x - 6 = -3 \Rightarrow 3x = 3 \Rightarrow x = 1\). L'antécédent de −3 est 1.

Exercice 8

La consommation électrique d'un atelier est modélisée par \(C(t) = 0{,}8t + 2\) (en kWh), où \(t\) est la durée de fonctionnement en heures.

  1. Quelle est la consommation après 5 h de fonctionnement ?
  2. Après combien d'heures la consommation atteint-elle 10 kWh ?
  1. \(C(5) = 0{,}8 \times 5 + 2 = 4 + 2 = 6\) kWh.
  2. On cherche l'antécédent de 10 : \(0{,}8t + 2 = 10 \Rightarrow 0{,}8t = 8 \Rightarrow t = 10\) h.

Exercice 9

Soit \(h(x) = 5x + 10\). Déterminer tous les antécédents de 40.

\(5x + 10 = 40 \Rightarrow 5x = 30 \Rightarrow x = 6\)
L'antécédent de 40 par \(h\) est 6 (unique, car \(h\) est une fonction affine).

C4 — Construire et compléter un tableau de valeurs

Rappel de cours

Un tableau de valeurs liste des valeurs de \(x\) (1ère ligne) et les images \(f(x)\) correspondantes (2e ligne). Pour chaque \(x\), on substitue dans la formule et on calcule.

Exercice 10

Compléter le tableau de valeurs de \(f(x) = 2x - 1\) pour \(x \in \{-2, -1, 0, 1, 2, 3\}\).

\(x\)−2−10123
\(f(x)\)
\(x\)−2−10123
\(f(x)\)−5−3−1135
Calculs : \(f(-2) = 2\times(-2)-1 = -5\) ; \(f(0) = -1\) ; \(f(3) = 5\).

Exercice 11

Le coût de location d'un échafaudage est \(C(j) = 25j + 40\) €, où \(j\) est le nombre de jours. Construire le tableau de valeurs pour \(j \in \{1, 2, 3, 5, 7\}\).

Jours \(j\)12357
Coût \(C(j)\) (€)6590115165215
Calculs : \(C(1) = 25 + 40 = 65\) ; \(C(5) = 125 + 40 = 165\).

Exercice 12

Voici un tableau de valeurs d'une fonction \(g\). Compléter les valeurs manquantes sachant que \(g(x) = x^2 + 1\).

\(x\)−3−1024
\(g(x)\)25
\(g(-3) = (-3)^2 + 1 = 10\)
\(g(-1) = 1 + 1 = 2\) ✔
\(g(0) = 0 + 1 = 1\)
\(g(2) = 4 + 1 = 5\) ✔
\(g(4) = 16 + 1 = 17\)
\(x\)−3−1024
\(g(x)\)1021517

C5 — Lire une courbe représentative

Rappel de cours

Sur un graphique : l'image de \(a\) se lit en partant de \(x = a\) sur l'axe horizontal, puis en montant jusqu'à la courbe, puis en lisant l'ordonnée. Pour trouver un antécédent de \(b\), on part de \(y = b\) sur l'axe vertical et on cherche les points de la courbe à cette hauteur.

Exercice 13

On considère la courbe d'une fonction \(f\) qui passe par les points suivants : \(A(0\,;\,2)\), \(B(1\,;\,4)\), \(C(2\,;\,6)\), \(D(3\,;\,8)\).

  1. Quelle est l'image de 2 par \(f\) ?
  2. Quel est l'antécédent de 4 par \(f\) ?
  3. Quelle est la valeur de \(f(0)\) ?
  4. Pour quelle valeur de \(x\) a-t-on \(f(x) = 8\) ?
  1. \(f(2) = 6\) (point C)
  2. L'antécédent de 4 est \(x = 1\) (point B)
  3. \(f(0) = 2\) (point A)
  4. \(f(x) = 8\) pour \(x = 3\) (point D)

Exercice 14

La courbe d'une fonction \(g\) atteint son maximum en \(x = 2\) avec \(g(2) = 9\), et passe par \(g(0) = 1\) et \(g(4) = 1\).

  1. Quelle est la valeur maximale de \(g\) ? En quel point est-elle atteinte ?
  2. Les valeurs \(g(0)\) et \(g(4)\) sont-elles égales ? Que peut-on supposer sur la symétrie de la courbe ?
  3. Donner un antécédent de 1 par \(g\).
  1. La valeur maximale est 9, atteinte en \(x = 2\).
  2. \(g(0) = g(4) = 1\) : les deux valeurs sont égales. La courbe semble symétrique par rapport à la droite \(x = 2\).
  3. Deux antécédents de 1 : \(x = 0\) et \(x = 4\).

Exercice 15

Une fonction \(f\) est définie par son tableau de valeurs :

\(x\)−2−10123
\(f(x)\)7434712
  1. Calculer \(f(-2) + f(2)\).
  2. Sur quel intervalle \(f\) semble-t-elle décroissante ?
  3. Quel semble être le minimum de \(f\) et en quel point ?
  4. Combien de valeurs de \(x\) ont pour image 4 ?
  1. \(f(-2) + f(2) = 7 + 7 = 14\)
  2. \(f\) semble décroissante sur \([-2\,;\,0]\) (les valeurs diminuent : 7, 4, 3).
  3. Le minimum apparent est 3, atteint en \(x = 0\).
  4. Deux valeurs : \(x = -1\) et \(x = 1\) donnent toutes deux \(f(x) = 4\).

C6 — Exploiter l'équation \(y = f(x)\) : appartenance d'un point

Rappel de cours

La courbe d'équation \(y = f(x)\) est l'ensemble des points \(M(x\,;\,y)\) tels que \(y = f(x)\). Pour vérifier qu'un point \(A(a\,;\,b)\) appartient à la courbe, on calcule \(f(a)\) : si \(f(a) = b\), le point est sur la courbe ; sinon, il n'y est pas.

Exercice 16

Soit \(f(x) = 2x^2 - 3x + 1\). Les points suivants appartiennent-ils à la courbe de \(f\) ?

  1. \(A(0\,;\,1)\)
  2. \(B(2\,;\,3)\)
  3. \(C(1\,;\,0)\)
  4. \(D(-1\,;\,4)\)
On vérifie si \(f(x_A) = y_A\) pour chaque point :
  1. \(f(0) = 0 - 0 + 1 = 1 = y_A\) → A est sur la courbe ✔
  2. \(f(2) = 8 - 6 + 1 = 3 = y_B\) → B est sur la courbe ✔
  3. \(f(1) = 2 - 3 + 1 = 0 = y_C\) → C est sur la courbe ✔
  4. \(f(-1) = 2 + 3 + 1 = 6 \neq 4 = y_D\) → D n'est pas sur la courbe ✘

Exercice 17

Un menuisier modélise le coût de production (en €) par \(C(x) = 5x + 80\) où \(x\) est le nombre de pièces.

  1. Le point \(P(10\,;\,130)\) est-il sur la courbe de \(C\) ? Interpréter.
  2. Le point \(Q(20\,;\,200)\) est-il sur la courbe de \(C\) ? Interpréter.
  3. Trouver les coordonnées du point de la courbe dont l'abscisse est 15.
  1. \(C(10) = 50 + 80 = 130 = y_P\) → P est sur la courbe. Produire 10 pièces coûte 130 €.
  2. \(C(20) = 100 + 80 = 180 \neq 200\) → Q n'est pas sur la courbe. Produire 20 pièces ne coûte pas 200 € mais 180 €.
  3. \(C(15) = 75 + 80 = 155\). Le point est \((15\,;\,155)\).
