Interrogation — Ch04 · Équations du premier degré CAP · 20 min · /10
🎯 Objectifs du chapitre cliquer pour développer
  • Tester si un nombre est solution d'une équation
  • Résoudre une équation du type ax + b = cx + d
  • Mettre en équation un problème simple
  • Vérifier la cohérence d'une solution dans le contexte

Socle

Résous les équations (les étapes sont guidées) :

1. (3 pts) \(x + 9 = 15\)

\(x = 15 - … = …\)

2. (3 pts) \(4x = 28\)

\(x = \dfrac{28}{…} = …\)

3. (4 pts) \(3x + 5 = 17\)

Étape 1 : \(3x = 17 - 5 = …\)

Étape 2 : \(x = \dfrac{…}{3} = …\)

Correction :

1. x = 6

2. x = 7

3. 3x = 12 → x = 4

Standard

1. (4 pts) Résous les équations :

  • \(4x - 7 = 2x + 9\)
  • \(3(x - 2) = 18\)

2. (3 pts) Un menuisier pose des panneaux. Pour un chantier, il utilise \(n\) panneaux standards (3 kg chacun) et 4 panneaux renforcés (7 kg chacun). Le total est 35 kg. Pose l'équation et résous-la.

3. (3 pts) Résous l'inéquation \(5x - 8 \geq 12\). Interprète le résultat dans un contexte de production (invente un contexte).

Correction :

1. 4x−7 = 2x+9 → 2x = 16 → x = 8 | 3x−6 = 18 → 3x = 24 → x = 8

2. 3n + 4×7 = 35 → 3n + 28 = 35 → 3n = 7 → n = 7/3 ≈ 2,33 → soit 2 panneaux standards (valeur entière)

3. 5x ≥ 20 → x ≥ 4 → ex: "il faut produire au moins 4 meubles"

Approfondissement

1. (4 pts) Résous le système :

\(\begin{cases} 2x + 3y = 16 \\ x - y = 2 \end{cases}\)

2. (3 pts) Un artisan facture 55 €/h pour les déplacements et 40 €/h pour le travail en atelier. Sur 8 h totales, sa recette est de 365 €. Combien d'heures a-t-il passé en déplacement ?

3. (3 pts) Résous et représente graphiquement l'ensemble des solutions de \(2x + 1 < 3x - 5\). Quelle est la valeur critique ?

Correction :

1. x = y+2 → 2(y+2)+3y = 16 → 5y+4 = 16 → y = 2,4 → x = 4,4

2. Soit d les heures de déplacement. 55d + 40(8−d) = 365 → 15d + 320 = 365 → 15d = 45 → d = 3 h

3. 2x+1 < 3x−5 → 6 < x → x > 6. Valeur critique : x = 6.
