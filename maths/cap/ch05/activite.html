Situation professionnelle — Coût de production dans un atelier de signalétique

Un atelier de signalétique produit des panneaux publicitaires. Les charges fixes (loyer, électricité, amortissement des machines) s'élèvent à 400 € par mois. Le coût variable est de 25 € par panneau (matières premières + main d'œuvre). Le gérant veut prévoir le coût total de production selon le nombre de panneaux fabriqués.

Problématique : Comment prévoir le coût total pour n'importe quel nombre de panneaux ? Peut-on représenter graphiquement cette relation ?

Partie 1 — Tableau de valeurs

Question 1 : Complète le tableau du coût total \(C(n)\) en fonction du nombre de panneaux \(n\) :

Question 2 : On appelle cette relation une fonction. On la note :

\[C(n) = 25n + 400\]

Que représente le 25 ? Que représente le 400 ?

25 = coût par panneau (coût variable) | 400 = charges fixes (même si on produit 0 panneau)

Partie 2 — Représentation graphique

Question 3 : Place les points du tableau dans le repère ci-dessous. Relie les points pour tracer la courbe.

Décrire la forme de la courbe : est-ce une droite ou une courbe ?

La représentation est une droite. C'est toujours le cas pour une fonction affine (ax+b).

Partie 3 — Sens de variation

Question 4 : Quand n augmente, C(n) augmente-t-elle ou diminue-t-elle ? Justifie à partir du tableau ou de la formule.

C(n) est croissante : quand n augmente, le terme 25n augmente, donc C(n) augmente.

Question 5 : Pour quel nombre de panneaux le coût total atteint-il 650 € ? Résoudre \(C(n) = 650\).

\(25n + 400 = 650\) → \(25n = …\) → \(n = …\) panneaux

25n = 250 → n = 10 panneaux

Bilan — Ce que j'ai appris

  • Une fonction associe à chaque valeur x une valeur f(x) (l'image).
  • Une fonction affine s'écrit \(f(x) = ax + b\) : sa représentation est une droite.
  • La constante b est l'ordonnée à l'origine (valeur pour x=0).
  • Une fonction est croissante si les images augmentent quand x augmente.
  • Pour trouver x tel que f(x) = valeur donnée, on résout une équation.
