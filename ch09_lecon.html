Cours – Géométrie dans le triangle

1. Théorème de Pythagore

Dans un triangle rectangle en A :
\[ BC^2 = AB^2 + AC^2 \]Exemple : AB=3, AC=4 → BC=\(\sqrt{9+16}=\sqrt{25}=5\)

2. Réciproque de Pythagore

Si \(BC^2 = AB^2 + AC^2\), alors le triangle est rectangle en A.

3. Trigonométrie (triangle rectangle)

Dans un triangle rectangle en A, pour l'angle B :
\[\cos B = \frac{\text{adjacent}}{\text{hypoténuse}} \quad \sin B = \frac{\text{opposé}}{\text{hypoténuse}} \quad \tan B = \frac{\text{opposé}}{\text{adjacent}}\]
Moyen mémothéchnique : SOH-CAH-TOA

4. Théorème de Thalès

Si DE ∥ BC, alors \(\dfrac{AD}{AB} = \dfrac{AE}{AC} = \dfrac{DE}{BC}\)
