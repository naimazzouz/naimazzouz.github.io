Aide — Rappel des formules

Fonction volume
V(x) = x(30 − 2x)²
V(x) = 4x³ − 120x² + 900x
pour x ∈ ]0 ; 15[
Dérivée
V'(x) = 12x² − 240x + 900
V'(x) = 12(x − 5)(x − 15)
V'(x) = 0 pour x = 5 ou x = 15
Maximum
V'(x) > 0 sur ]0 ; 5[ → croissant
V'(x) < 0 sur ]5 ; 15[ → décroissant
Maximum en x = 5 : V(5) = 2 000 cm³
Dimensions du bac
Hauteur : x = 5 cm
Longueur = Largeur : 30 − 2×5 = 20 cm
Volume : 5 × 20 × 20 = 2 000 cm³
